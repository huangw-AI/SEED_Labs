# SEED_Labs
SEED Labs at UOW

This lab was developed and conducted using the Ubuntu 20.04 SEED virtual machine (VM). You can download a pre-built image from the University CSE network labs and run it on the lab computers, or you may download and run the SEED VM on your own computer.
https://github.com/seed-labs/seed-labs

## Acknowledgements
•	The lab exercises are conducted as part of the world-renowned SEED Lab Project for cybersecurity education.
•	The University of Westminster, as one of the participating international institutions, along with its instructors and students, has permission to access these resources.
•	The teaching materials are modified from the SEED Project. All materials provided in this course are intended strictly for educational purposes only.

## Safety and Ethics Note 
•	This lab uses intentionally vulnerable systems solely for educational purposes.
•	All activities must be conducted within the university local lab virtual machine (VM) environment. Testing against external or production systems is strictly prohibited.
